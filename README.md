# Library Management System

This project implements a Library Management System for the Técnicas de Programación final.

## Features
- Load inventory from CSV
- Keep a general list and an ordered list (by ISBN) maintained with insertion
- Linear search (title / author)
- Binary search (ISBN)
- Merge sort report by book value
- Shelving module: brute force and backtracking
- Stacks for user loan history and queues for reservations

## How to run
1. Make sure you have Python 3.8+
2. Create the folder structure shown in the document
3. Place `libros.csv` into `data/`
4. Run `python main.py`
